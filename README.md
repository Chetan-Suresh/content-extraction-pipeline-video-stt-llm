# Content Extraction Pipeline Using Speech-to-Text and Large Language Models

An end-to-end backend pipeline that extracts structured content from video data by converting speech into text and processing it with large language models.  
The system is demonstrated using cooking videos, where spoken instructions are transformed into structured recipes.

---

## Overview

This project processes video content through multiple stages:
1. Audio extraction from video sources (e.g., YouTube)
2. Speech-to-text transcription
3. Content understanding and structuring using large language models
4. Clean JSON output exposed via an API

The focus of this project is backend architecture, AI integration, and clean system design.
